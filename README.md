# Base58 ID

Create random base58 ID's based on UUID's.

## Motivation

I wanted the simplest possible way of generating unique base58 ID's.

## Usage

```
id := b58iud.New() // e.g. "A3hYyNE2ZXNFJZbTenaB36"
```
